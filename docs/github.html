Rising Sun

The program has a filled circle which is at the bottom of the window that it opens with, The instructions at the top left say to click the mouse repeatedly,
when clicked on multiple times the circle moves up slightly and the instructions dissapear. When the mouse exits the window the program resets as if nothing has been done.

This is a test of BlueJ and its functionality with javascript. I used commands like "object.draw" "set.color", and "onMouseClick/Exit", demonstrating how to create events from mouse input using js.
